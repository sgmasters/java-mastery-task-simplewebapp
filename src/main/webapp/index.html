Employees

Practical task for Godel Technologies

In this task, I used Spring Boot, and during the study, it became interesting for me to try all its features, so I implemented Dao first using the @Repository annotation and inherited from the Spring Data JpaRepository already implemented. Then to comply with the task implemented jdbcDao in a separate branch: spring_jdbc_dao_implementation.

EmployeeService essentially only delegates the execution of Dao requests to the class. An important point was the work with the Date type in PostgreSQL. I even had to create an additional Converter class from java LocalDate to Date type in DB.

After completing the main part of the task, I took up the unit tests. Implemented a test for SpringBootApplication that checks the loading of the application context. In addition, a test for a REST controller. The test checks the performance of all available CRUD operations.

I tried to write my UI using Angular framework, but look like it needs too much additional efforts. So I decided to write requests via Ajax.
